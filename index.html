HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading elements are represented in levels from h1-h6.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists.
  • The img element contains information about images that are displayed on the webpage.
  • The br element creates a line break between two blocks of content.
  • The a element creates links to webpages via the href attribute.

CSS

  • The style attribute can be used inline or as an internal style sheet, but the most common way to style a webpage is by using an external CSS style sheet.
  • The selector defines the element or attributes to which the rules, or declarations, will apply.
  • The * selector is used to apply a rule to all the elements visible on the page.
  • Selectors can be combined into a single CSS rule with a comma.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • The margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.
  • CSS style sheets must be linked to the HTML code to go into effect.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • The console.log method outputs a message to the web console by adding an argument.
  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • We can use conditional statements and loops to control the order in which code is executed.
  • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false.
  • All other values, including a Boolean with the value of true, are considered truthy.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
