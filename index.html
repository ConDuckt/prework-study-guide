Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading elements are represented in levels from h1-h6.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists.
  • The img element contains information about images that are displayed on the webpage.
  • The br element creates a line break between two blocks of content.
  • The a element creates links to webpages via the href attribute.

CSS

  • The style attribute can be used inline or as an internal style sheet, but the most common way to style a webpage is by using an external CSS style sheet.
  • The selector defines the element or attributes to which the rules, or declarations, will apply.
  • The * selector is used to apply a rule to all the elements visible on the page.
  • Selectors can be combined into a single CSS rule with a comma.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • The margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.
  • CSS style sheets must be linked to the HTML code to go into effect.
